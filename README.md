# The Coin Shack

Private workspace for taking over The Coin Shack — a physical
BTC/cash exchange shop in Markham, Ontario, Canada.

This is not the trading app. It's notes, prototypes, and small
web widgets that support the takeover and ongoing operations.
The revamped trading app will live in its own repo when that
work starts.

Planned layout (built out incrementally by per-repo AI sessions):
- `notes/` — training notes from current operators, sporadic
  process documentation, raw thinking. Never deploys.
- `corpus/` — regulator references, vendor manuals, sanitized
  snippets of the existing trading app being analyzed.
- `src/` — prototype code, tooling, small utilities.
- `site/` — small web widgets shared with the inherited team
  (brother, wife, employee) via an allowlist-controlled deploy.

See `VISION.md` for the why, `HANDOFF.md` for AI session context.
